Android Tutorial (Termux)
1Install Termux
Download Termux from Google Play Store or F-Droid
2Update packages
pkg update && pkg upgrade
3Install Python
pkg install python
4Download Kiffy
pkg install wget && wget https://kiffy-msg.vercel.app/kiffy_client.py
5Run Kiffy
python kiffy_client.py
